## 3. 역할별 접근 화면 매핑

| 액터/역할 | 접근 가능 SCR 목록 | 비고 |
|:---|:---|:---|
| **비인증 사용자** | SCR-001 | 로그인 화면만 접근. 그 외 모든 경로는 `/login`으로 리다이렉트 |
| **보안담당자** | SCR-001, SCR-002, SCR-003, SCR-004, SCR-005, SCR-006, SCR-007, SCR-008, SCR-009, SCR-010 | SCR-011(설정) 접근 불가. 초기화 버튼 비활성화 |
| **관리자** | SCR-001 ~ SCR-011 (전체) | 모든 화면 접근 가능. SCR-011 단독 접근 권한 |

### 접근 통제 구현 방식

```
middleware.ts
  └── 세션 없는 요청 → /login 리다이렉트
      세션 있는 요청 → role 확인
          role === 'user'  → /settings 접근 시 403 반환
          role === 'admin' → 전체 접근 허용
```
