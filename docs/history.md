● AI-DLC 요구사항 정의서 작성기

  사용자의 자연어 입력을 분석해 AI-DLC 방법론 표준 목차의 한국어 요구사항 정의서를 생성합니다.

  ---
  사용 방법

  자연어로 사업 내용을 입력하면 됩니다. 예:

  사내 출퇴근 관리 시스템. QR 출퇴근 체크, 휴가 신청, 관리자 대시보드.
  동시접속 300명, JWT 인증. FE는 React, BE는 Spring Boot, DB는 PostgreSQL.

  ---
  진행 절차

  1. 입력 파싱 — 사업명, 기능/비기능 요구사항, 기술스택 자동 추출
  2. 초안 생성 — 표준 템플릿 기반으로 즉시 작성
  3. 핵심 누락 항목 묶음 질문 — 한 번만, 3개 이하로 추려서 질문
  4. 최종 파일 저장 — 요구사항정의서_{사업명}_{YYYYMMDD}.md
