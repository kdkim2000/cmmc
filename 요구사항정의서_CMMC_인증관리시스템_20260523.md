## 1. 사업개요

### 1) 사업목적

CMMC(Cybersecurity Maturity Model Certification) Level 1 및 Level 2 인증 달성을 위해 각 점검항목의 AS-IS 현황을 체계적으로 확인·기록하고, 갭(Gap) 분석 결과를 바탕으로 보완 계획(POA&M)을 수립하여 본 심사까지 진행 상황을 추적 관리할 수 있는 웹 애플리케이션 시안을 구축한다.

### 2) 사업개요

본 시스템은 CMMC Level 1(FAR 52.204-21, 15개 요구사항/58개 평가목표)과 CMMC Level 2(NIST SP 800-171A, 14개 도메인/110개 요구사항/320개 평가목표)의 점검항목을 웹 화면에서 직접 평가하고, 미충족(NOT MET) 항목에 대한 보완 계획을 등록·추적할 수 있는 기능을 제공한다. 3일 내 AI-DLC 방법론을 활용하여 빠르게 개발 시안을 완성하고, Vercel 플랫폼에 배포하여 검토한다.

### 3) 추진 배경

국방부 계약(DoD 계약) 및 미 연방정부 공급망 참여를 위해서는 FCI(연방계약정보) 취급 조직은 CMMC Level 1, CUI(통제 비분류 정보) 취급 조직은 CMMC Level 2 인증을 의무적으로 획득해야 한다. 현재 조직 내 보안 통제 현황을 체계적으로 파악하고, 심사 전 자체 점검·보완 이력을 관리할 수 있는 전용 도구가 필요한 상황이다.

### 4) 사업 내용

- CMMC Level 1 (15개 요구사항 / 58개 평가목표) 점검 화면 구현
- CMMC Level 2 (14개 도메인 / 110개 요구사항 기반) 점검 화면 구현
- 각 항목별 MET / NOT MET 판정 입력 및 저장
- 미충족 항목 기반 갭 분석 결과 화면 제공
- 보완 계획(POA&M) 등록 및 진행 상태 추적
- 증적(Artifact) 참조 정보 등록 (파일명 / URL / 비고)
- 전체 준비 현황 대시보드 및 SPRS 점수 시뮬레이터 (Level 2)
- Next.js 풀스택 + Neon Postgres (Vercel Storage) 기반으로 Vercel 배포

## 3. 기술 스택 및 인프라

### 1) 기술스택

| 계층 | 기술 | 버전/비고 |
|:---|:---|:---|
| Frontend | Next.js (App Router) + React + TypeScript | Next.js 15, React 19 |
| UI | Tailwind CSS + shadcn/ui | 반응형 컴포넌트 라이브러리 |
| Backend | Next.js API Routes / Server Actions (Node.js) | 별도 백엔드 서버 없음, Next.js 풀스택 |
| Database | Neon (Vercel Postgres, serverless PostgreSQL) | Vercel 대시보드 1클릭 연동, 무료 플랜 0.5GB |
| ORM | Drizzle ORM | PostgreSQL adapter, 타입 안전성 |
| 인증 | NextAuth.js v5 (Auth.js) | 간단한 Credentials Provider |
| Infra/Cloud | Vercel | Hobby 플랜, Git 연동 자동 배포 |
| 기타 | Recharts 또는 Chart.js | 대시보드 차트 시각화 |

### 2) 인프라 구성

- GitHub 저장소에 Next.js 프로젝트를 관리하고, Vercel에 연결하여 `main` 브랜치 Push 시 자동 배포된다.
- DB는 Vercel 대시보드 → Storage 탭에서 Neon Postgres를 1클릭으로 연결하며, `DATABASE_URL` 환경변수가 Vercel 프로젝트에 자동 주입된다. 별도 외부 서비스 가입 없이 Vercel 계정 하나로 DB까지 통합 관리한다.
- 전체 인프라는 서버 운영 없이 Vercel + Neon의 관리형 서비스로만 구성한다.

```
[사용자 브라우저]
        │ HTTPS
        ▼
[Vercel Edge / Next.js App]
   ├── App Router Pages (React Server Components)
   ├── Server Actions / API Routes
        │ DATABASE_URL (자동 주입)
        ▼
[Neon Postgres (Vercel Storage)]
   └── evaluations / poa_and_m / artifacts 테이블
```
