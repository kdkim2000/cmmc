## 제품 개요

| 항목 | 내용 |
|:---|:---|
| **제품명** | CMMC Compliance Management System |
| **목적** | CMMC Level 1/2 인증 달성을 위한 AS-IS 점검·갭분석·POA&M 추적 관리 |
| **대상 사용자** | 보안담당자, CMMC 심사 준비 관리자 |
| **개발 기간** | 2026-05-23 ~ 2026-05-25 (3일 MVP 시안) |
| **배포** | Vercel (Hobby) |

---

## 핵심 문제 정의

미 연방정부 DoD 계약 참여를 위해 CMMC 인증이 필수화되었으나, 조직 내 보안 통제 현황을 체계적으로 파악하고 심사 전 자체 점검·보완 이력을 관리할 전용 도구가 없다. 스프레드시트 기반 관리는 버전 충돌, 접근 권한 제한, 실시간 현황 파악 불가 등의 문제를 야기한다.

## 데이터 범위

| 데이터 | 규모 | 관리 방식 |
|:---|:---|:---|
| Level 1 점검항목 | 15개 요구사항 / 58개 평가목표 | 정적 JSON Seed |
| Level 2 점검항목 | 110개 요구사항 | 정적 JSON Seed |
| 평가 결과 | 항목수 × 평가 회차 | Neon Postgres |
| POA&M | NOT MET 항목 수 이하 | Neon Postgres |
| 증적 참조 | 항목당 N개 | Neon Postgres |

## 성공 기준
